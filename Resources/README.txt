AutoMatty Icon Resources
========================

Place your custom toolbar icon here:

  Icon_AutoMatty_40x.png
    - Size: 40x40 pixels
    - Format: PNG with transparency
    - Style: Should match Unreal's toolbar icon style (light on dark)

Recommended Icon Design:
  - Use white or light gray for the main icon
  - Keep it simple and recognizable at small sizes
  - Add a subtle shadow or outline for better visibility
  - Test it against both dark and light backgrounds

If no custom icon is provided, the plugin will fall back to Unreal's default icon.

You can find free icon resources at:
  - Font Awesome (https://fontawesome.com/)
  - Material Icons (https://fonts.google.com/icons)
  - Unreal's built-in icon set in Engine/Content/Slate

To test your icon:
  1. Place Icon_AutoMatty_40x.png in this folder
  2. Run BuildPlugin.bat
  3. Restart Unreal Editor
  4. Check the toolbar next to the Play button
